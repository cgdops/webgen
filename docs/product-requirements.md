# Product Requirements: webgen

## 1. App Overview
webgen is a minimal website management app designed for single users to build and manage personal and company websites easily and with no code and little to no technical experience.

## 2. Target Audience
Individual users who need a simple website and a way to gather leads from it.

## 3. Core Features (MVP - Minimum Viable Product)
1. **User Authentication:**
   - Users can sign up with email/password.
   - Users can log in and log out.
2. **Website Creation:**
   - Users can connect their linkedin profile to create a new website.
   - Websites are created by coding agents following content in the linkedin profile.
3. **Website Management:**
   - Users can click on a website to open the website details.
   - Users can deploy their website to GitHub Pages in 1 click.
4. **Website Viewing:**
   - Users can see a list of all active websites.
   - Users can see contact details created by the contact form.
   - Users can click a link to view their live github page.

## 4. Out of Scope (Do NOT Build)
- No linkedin company pages
- No website analytics or tracking tools
- No Content Management or content creation tools
- No "Dark Mode" (keep UI strictly black/white for now).
